What's new

Updates, improvements, and fixes to CheckGen.

March 2026
Launch
v2.0

CheckGen v2 — Accounts & Cloud History

The biggest update yet. CheckGen v2 adds user accounts, cloud-saved history, and a redesigned interface built to scale.

  • Accounts — Create a free account with email/password or Google SSO. Your profile, history, and sessions are tied to your account.
  • Cloud history — Saved checklists sync across devices. Pick up exactly where you left off from any browser.
  • Team page — Start shared sessions and see your organization. Full team management (invite links, roles, shared history) is coming soon.
  • Left navigation — New sidebar in the app for New Checklist, History, Team, and Account.
  • Redesigned landing page — New homepage, about page with FAQ, pricing, terms, privacy policy, and changelog.
  • Password requirements — Minimum 8 characters, one uppercase, one number, one special character.
  • Anonymous limit — Guests can generate 3 checklists before being prompted to create an account.
  • Outcome buttons — Pass, Fail, Blocked now display horizontally with full labels.
  • Green design system — Emerald green replaces indigo as the primary brand color throughout.
Feb 2026
Update
v1.2

Viewport scaling & polish

  • Responsive text — All font sizes now use clamp() to scale across screen sizes from 13" to 27" monitors.
  • History improvements — Clear all history button, P/F/B stats in history cards, live sync between active session and history entry.
  • End session reset — Ending a session now fully clears all form fields, selects, and checkboxes.
  • Back button guard — Going back to edit AC while testing now warns before discarding progress.
  • Shared session error handling — If Supabase is unavailable, the session falls back gracefully to personal mode.
Jan 2026
Update
v1.1

Notes, CSV export, and time estimates

  • Notes per item — Add notes to individual test steps. Notes persist in localStorage and export with CSV.
  • CSV export — Filter by outcome (all, no outcome, has outcome) and export. Notes, priority, type, and time estimate included per row.
  • Time summary — Total estimated time for the session shown as a pill before you start.
  • Add custom steps — Add your own test steps to any section of the checklist.
  • Delete items — Remove individual items from the checklist.
  • Regen by section — Regenerate just one section without touching the rest.
  • Shared sessions — Real-time P/F/B sync via Supabase with a 6-character join code.
Dec 2025
Launch
v1.0

Initial launch

CheckGen started as an internal QA tool — a way to stop writing the same test cases from scratch every sprint.

  • AI generation — Paste AC, select testing areas, get 15–25 grouped test cases from Claude Haiku.
  • Grouped checklist — Items organized by section (Functional, Permissions, Validation, Edge Cases, etc.).
  • Pass / Fail / Blocked — Mark outcomes per item as you test.
  • Progress bar — Live count of actioned vs total items.
  • localStorage persistence — Session survives page refresh.
  • Ticket ID, environment, checklist name — Optional metadata fields shown in the checklist header.
